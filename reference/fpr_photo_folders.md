# Build directories where the photos will be placed based on a list of site ids

Build directories where the photos will be placed based on a list of
site ids

## Usage

``` r
fpr_photo_folders(site, path = "data/photos/")
```

## Arguments

- site:

  String - number of site to build folder for

- path:

  String - path stub where photos directories are held

## Value

Makes directories
