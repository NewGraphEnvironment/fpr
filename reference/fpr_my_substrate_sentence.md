# Sentence to describe the dominant and sub-dominant substrate

Sentence to describe the dominant and sub-dominant substrate

## Usage

``` r
fpr_my_substrate_sentence(loc = "us", sit = my_site)
```

## Arguments

- loc:

  String in quotes which is either 'us' upstream or 'ds' downstream.
  Defaults to 'us'

- sit:

  Integer. Corresponds to the PSCIS site id. Defaults to my_site defined
  in environment

## Value

String sentence with the cover types described
