# Resize images and convert JPG

Resize images and convert JPG

## Usage

``` r
fpr_photo_resize_convert(photo, size = "1036.8x777.6!", path = NULL)
```

## Arguments

- photo:

  String of full path name of photo

- size:

  String image width and height of photo you want to resize to

- path:

  String of path to the directory where you want to burn the converted
  img
