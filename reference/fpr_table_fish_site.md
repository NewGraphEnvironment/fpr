# Summary table for fish sampling sites

Summary table for fish sampling sites

## Usage

``` r
fpr_table_fish_site(dat = tab_fish_sites_sum, sit = my_site, ...)
```

## Arguments

- dat:

  Dataframe. Defaults to tab_fish_sites_sum. Must have columns named
  site, passes, ef_length_m, ef_width_m, area_m2, enclosure

- sit:

  Integer. Site ID. Defaults to my_site

- ...:

  Not used. Open for passing arguments to
  [fpr_kable](http://www.newgraphenvironment.com/fpr/reference/fpr_kable.md)

## Value

Dataframe
