# Resize batch of images and convert JPG

Resize batch of images and convert JPG

## Usage

``` r
fpr_photo_resize_batch(dir_source = NULL, dir_target = NULL, ...)
```

## Arguments

- dir_source:

  String full path name of directory where photos are at

- dir_target:

  String full path name of directory where photos are to be copied to

- ...:

  Not used. For passing alternative destination photo size as string to
  [fpr_photo_resize_convert](http://www.newgraphenvironment.com/fpr/reference/fpr_photo_resize_convert.md)

## Value

Photos resized with \`.JPG\` extension
