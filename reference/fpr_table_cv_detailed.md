# Make two column dataframe of PSCIS info with parameter on left and value on right within each column.

Make two column dataframe of PSCIS info with parameter on left and value
on right within each column.

## Usage

``` r
fpr_table_cv_detailed(dat)
```

## Arguments

- dat:

  Datframe of PSCIS spreadsheet input data with only 1 row. Split from
  pscis_phase1, pscis_reasessments or pscis_phase2.

## Value

Two column dataframe with parameter on left and value on right within
each column.
