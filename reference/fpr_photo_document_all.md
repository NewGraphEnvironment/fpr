# Document all the photos that are in the directories to csv as a bay to track changes after scripted organization

Document all the photos that are in the directories to csv as a bay to
track changes after scripted organization

## Usage

``` r
fpr_photo_document_all(path_to_photo_dir, full_names = T)
```

## Arguments

- path_to_photo_dir:

  Full path to photo directories to be

- full_names:

  whether we should use full names when listing photos
