# Make table for the comments to be appended to

Make table for the comments to be appended to

## Usage

``` r
fpr_table_cv_detailed_comments(dat)
```

## Arguments

- dat:

  Dataframe of PSCIS spreadsheet input data with only 1 row. Split from
  pscis_phase1, pscis_reasessments or pscis_phase2.

## Value

Dataframe two columns wide with comments: on the left
