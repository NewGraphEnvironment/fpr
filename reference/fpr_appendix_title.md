# Build the title for the appendices

Build the title for the appendices

## Usage

``` r
fpr_appendix_title(site = my_site, site2 = NULL, site3 = NULL)
```

## Arguments

- site:

  Numeric value of site. Usually PSCIS ID. Defaults to object my_site.

- site2:

  Numeric value of site 2. Defaults to NULL.

- site3:

  Numeric value of site 3. Defaults to NULL.
