# Sentence to describe survey length of your site.

No hard stop at the end of the sentence so we can reference the photos.

## Usage

``` r
fpr_my_survey_dist(loc = "us", sit = my_site)
```

## Arguments

- loc:

  String in quotes which is either 'us' upstream or 'ds' downstream.
  Defaults to 'us'

- sit:

  Integer. Corresponds to the PSCIS site id. Defaults to my_site defined
  in environment

## Value

String sentence with the direction and distance surveyed.
